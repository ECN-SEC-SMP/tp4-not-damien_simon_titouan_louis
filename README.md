# TP4 - Gestion de Parcelles Cadastrales

**Auteurs :** Louis VAILLANT, Damien BUREAU, Titouan COPIN, Simon CAU

## Description

Système de gestion de parcelles en C++ utilisant l'héritage simple, multiple et virtuel pour modéliser différents types de zones (urbaines, naturelles, agricoles, à urbaniser).

Documentation complète : [COMPTE_RENDU.md](doc/COMPTE_RENDU.md)

## Compilation

Projet compilable avec CMake.
